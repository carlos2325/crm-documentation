Autenticación con GraphQL

Aprende cómo autenticarte en la API GraphQL, y luego cómo crear y ejecutar consultas y mutaciones.

Configurar Headers

const headers = {
  'Content-Type': 'application/json',
  'Authorization': 'Bearer YOUR_JWT_TOKEN',
  'x-whitelabel-id': 'YOUR_WHITELABEL_ID'
};

Primera Query

query GetLeads {
  crmLeads {
    id
    name
    email
    status
    createdAt
  }
}

Ejecutar Request

const response = await fetch('https://api2.eventosorganizador.com/graphql', {
  method: 'POST',
  headers,
  body: JSON.stringify({ query })
});
